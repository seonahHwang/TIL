# lesson 24 : TCP의 구조

TCP 헤더 : TCP로 전송할 때 데이터에 붙이는 헤더

세그먼트 : TCP 헤더가 붙은 데이터 

데이터를 전송하기 전에 연결을 확보해야함!

TCP 헤더의 코드 비트(6비트)에 SYN ACK FIN 등이 있음

SYN : 연결 요청

ACK : 확인 응답

## 3 way handshake

연결 확립을 위해 패킷 교환을 세번 하는 것

SYN, ACK를 이용하여 3 way handshake를 함

1. 연결 확립 요청

    A → B (SYN)

2. 연결 확립 응답, 연결 확립 요청

    B → A (SYN, ACK)

3. 연결 확립 응답

    A → B (ACK)

연결 종료시에는 FIN, ACK를 사용
